# SolaPay Backend

This is the backend server for the SolaPay application, providing APIs for user authentication, invoice creation, notifications, and blockchain interactions on the Solana devnet. Built using Node.js, Express, MongoDB, and integrated with the Solana Web3 and Metaplex SDKs.

## Features

- OAuth-based authentication (Google, Discord) using Passport.js
- REST API for invoice management (create, fetch, update, delete)
- Solana Pay payment tracking and USDC/SOL wallet funding (devnet)
- MongoDB-based user and invoice storage
- Email notifications using Nodemailer
- In-app notification system
- QR code generation for payment links

## Technologies Used

- Node.js
- Express.js
- TypeScript
- MongoDB + Mongoose
- Passport.js (OAuth)
- Solana Web3.js
- Metaplex SDK
- Nodemailer
- QR Code Generator
